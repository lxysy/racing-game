# Three.js 赛车游戏设计文档

参考游戏：Forza Horizon（地平线）系列  
技术栈：Three.js r175 + Vite + simplex-noise  
项目位置：`F:\code\racing-game\`

---

## 1. 游戏概述

第三人称追尾视角的 3D 赛车游戏，在环形赛道上驾驶，感受速度与风景。

## 2. 技术方案

| 项目 | 方案 |
|------|------|
| 构建工具 | Vite（参考 snowy-forest 模式） |
| 3D 引擎 | Three.js r175 |
| 地形算法 | simplex-noise 程序化噪声 |
| 赛道定义 | CatmullRomCurve3 样条曲线 |
| 物理引擎 | 自建街机风格物理 |


## 4. 车辆模型

| 项目 | 内容 |
|------|------|
| 来源 | Sketchfab - Straight Design (creativemango) |
| 许可证 | CC-BY-SA-4.0（需署名 + 相同方式共享） |
| 文件 | `low_poly_car/scene.gltf` + `scene.bin` |
| 面数 | ~5000 面，适合游戏 |
| 格式 | GLTF，Three.js 原生支持 |

**节点结构：**
```
RootNode
├── WRR (右后轮) → 轮胎 + 轮毂    ✅ 独立节点，可旋转
├── WRL (左后轮) → 轮胎 + 轮毂    ✅ 独立节点，可旋转  
├── WFL (左前轮) → 轮胎 + 轮毂    ✅ 独立节点，可旋转
├── WFR (右前轮) → 轮胎 + 轮毂    ✅ 独立节点，可旋转
├── TailLight                      → 尾灯（后续可用作刹车灯）
└── CarBody → 6 个子网格           → 车身（含绿色迷彩）
```

四个轮子均为独立节点，可在游戏循环中单独旋转实现车轮动画。

## 5. 核心模块设计

### 5.1 赛道 (Track.js)

- 用 `CatmullRomCurve3` 定义一条环形赛道（8~12 个控制点）
- 生成道路：沿曲线生成带状路面，宽度约 8~12 单位
- 路肩：道路两侧加护栏或路肩标记
- 支持查询任意 t 值的位置和朝向
- 赛道为闭环，支持圈速计时

### 5.2 赛车 (Car.js)

**外观模型**: 导入 GLTF 模型（详见下方"车辆模型"章节）

**物理模型**（街机风格）:
- `speed`: 当前速度（标量）
- `acceleration`: 按住 W/↑ 加速
- `braking`: 按住 S/↓ 减速
- `steering`: A/D 或 ←/→ 改变方向
- `maxSpeed`: 最高速度限制
- `friction`: 自然减速模拟阻力
- 转弯半径与速度成反比（高速转向迟钝）
- 赛车沿赛道高度飞行（或跟随地形）

### 5.3 第三人称相机 (FollowCamera.js)

- 相对位置：车后 10~15 单位，上方 5~8 单位
- 使用 `lerp()` 平滑插值跟随
- 随速度增大，相机轻微后移（增强速度感）
- `lookAt` 始终指向赛车前方

### 5.4 环境 (Environment.js)

**地面/地形**:
- PlaneGeometry(300, 300, 80, 80)
- simplex-noise 生成高度起伏
- vertexColors 渐变（草绿 → 棕色）
- 赛道位置压低地形保证道路平整

**树木**:
- 简易树木（CylinderGeometry 树干 + ConeGeometry 树冠）
- 道路两侧随机分布
- 作为速度参照物

**天空/光照**:
- `scene.background` 天空蓝色渐变
- `DirectionalLight` + `AmbientLight`
- 方向光投射阴影增强立体感

### 5.5 HUD (HUD.js)

- 速度显示（km/h）
- 圈速计时（如果是闭环赛道）
- CSS overlay 实现

### 5.6 主循环 (main.js)

流程：
1. `init()`: 创建场景、相机、渲染器、光照
2. `animate()` 每帧:
   - 读取键盘输入 → `car.update(dt)`
   - `camera.update(car)`
   - `hud.update(car)`
   - `renderer.render(scene, camera)`
3. 窗口 resize 自适应

## 6. 控制方案

| 按键 | 功能 |
|------|------|
| W / ↑ | 加速 |
| S / ↓ | 刹车 / 倒车 |
| A / ← | 左转 |
| D / → | 右转 |

## 7. 实现步骤

1. 初始化 Vite 项目，安装 `three` 和 `simplex-noise`
2. 创建基础场景：地面、光照、天空
3. 实现赛道生成
4. 实现赛车外观 + 物理
5. 实现第三人称相机跟随
6. 添加环境装饰（树木、地形起伏）
7. 添加 HUD 速度显示
8. 调优：物理手感、相机位置、视觉风格

## 8. 运行方式

```bash
cd three-js-study/racing-game
npm install
npm run dev
```

然后打开浏览器访问即可驾驶。
